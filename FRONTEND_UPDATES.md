# 前端优化说明

## 更新内容

### 1. Token 列表显示优化

#### 新增列
- **状态列**: 显示 Token 的实时状态（正常/余额不足/账号封禁/速率限制等）
- **错误次数列**: 显示连续错误次数，超过 3 次会高亮警告

#### 状态徽章
- 🟢 **正常** (active): 绿色徽章，Token 可正常使用
- 🔴 **余额不足** (payment_required): 红色徽章，402 错误
- 🔴 **账号封禁** (forbidden): 红色徽章，403 错误
- 🟡 **速率限制** (rate_limited): 黄色徽章，429 错误，显示下次重试时间
- 🟠 **Token过期** (unauthorized): 橙色徽章，401 错误
- 🔴 **错误** (error): 红色徽章，其他错误

#### 状态提示
- 鼠标悬停在状态徽章上可查看详细错误信息
- 如果有下次重试时间，会显示具体的重试时间点

### 2. 额度检查反馈优化

#### 单个 Token 刷新
点击"刷新额度"按钮后，会显示详细信息：

**成功时**：
```
✅ 额度检查成功

📊 额度信息:
  总额度: 500,000
  已使用: 12,345
  剩余: 487,655
  订阅类型: plus

👤 账号信息:
  邮箱: user@example.com
  账号ID: user-xxx
  订阅: plus
```

**失败时**：
```
❌ 额度检查失败

状态: payment_required
错误: 账号余额不足或订阅已过期
错误码: 402

⚠️ 该 Token 已被自动禁用
原因: 账号余额不足或订阅已过期
```

#### 批量刷新
点击"刷新全部额度"按钮后，会显示汇总信息：

```
✅ 批量刷新完成

📊 统计:
  总数: 10 个
  成功: 7 个
  失败: 2 个
  自动禁用: 1 个

⚠️ 错误详情:
  • Token 3: 账号余额不足或订阅已过期 (已自动禁用)
  • Token 5: 请求频率超限
```

### 3. 视觉优化

#### 错误次数警告
- 错误次数 >= 3 时，显示为红色加粗并带有警告图标 ⚠️
- 使用动画效果（闪烁）吸引注意

#### 操作按钮图标化
- 刷新额度: 🔄 图标
- 启用/禁用: ▶️/⏸️ 图标
- 删除: 🗑️ 图标

#### 名称显示优化
- 优先显示 name，其次 email，再次 account_id
- 在名称下方显示启用/禁用状态徽章

### 4. 表格列调整

**调整前**：
```
名称 | 额度 | 总请求 | 成功 | 失败 | 过期时间 | 状态 | 操作
```

**调整后**：
```
名称 | 状态 | 额度 | 总请求 | 成功 | 失败 | 错误次数 | 最后使用 | 操作
```

**优化点**：
- 移除"过期时间"列（不太重要）
- 添加"状态"列（显示实时状态）
- 添加"错误次数"列（快速识别问题 Token）
- 将"状态"改为"最后使用"（更实用）

## 使用建议

### 快速识别问题 Token

1. **查看状态列**：
   - 红色徽章 = 需要立即处理
   - 黄色徽章 = 暂时限制，会自动恢复
   - 绿色徽章 = 正常

2. **查看错误次数**：
   - 0 = 正常
   - 1-2 = 偶尔失败，可能是网络问题
   - 3+ = 持续失败，需要检查（红色闪烁警告）

3. **查看额度**：
   - 绿色 = 额度充足
   - 黄色 = 已使用 50-80%
   - 红色 = 已使用 80% 以上

### 定期维护

1. **每天检查一次**：
   - 点击"刷新全部额度"
   - 查看有多少 Token 被自动禁用
   - 处理失败的 Token

2. **关注错误次数**：
   - 错误次数 >= 3 的 Token 需要重点关注
   - 可能需要刷新 Token 或更换账号

3. **及时处理失效 Token**：
   - 余额不足：充值或更换账号
   - 账号封禁：删除并添加新账号
   - 速率限制：等待重试时间后自动恢复

## 技术细节

### 状态映射

| 后端状态 | 前端显示 | 颜色 | 图标 |
|---------|---------|------|------|
| active | 正常 | 绿色 | ✓ |
| payment_required | 余额不足 | 红色 | 💳 |
| forbidden | 账号封禁 | 红色 | 🚫 |
| rate_limited | 速率限制 | 黄色 | ⏰ |
| unauthorized | Token过期 | 橙色 | 🔑 |
| error | 错误 | 红色 | ⚠️ |

### 自动刷新

- Token 列表会在以下操作后自动刷新：
  - 添加 Token
  - 删除 Token
  - 刷新额度
  - 启用/禁用 Token

### 响应式设计

- 表格支持横向滚动
- 在小屏幕上保持可读性
- 状态徽章自适应大小
